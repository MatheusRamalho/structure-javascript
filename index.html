Evento Recursos
cached // Os recursos listados no manifesto foram baixados, e a aplicação ja está armazenada em cache. error // Falha no carregamento do recurso. abort // O carregamento do recurso foi abortado load // O recurso e suas dependências foram carregadas beforeunload // Os recursos da página que estavam sendo baixados estão prestes a ser cancelados. unload // O documento ou suas dependências estão sendo canceladas.
Evento Rede
online // O navegador está conectado à rede. offline // O navegador perdeu acesso à redeFoco
Evento Foco
focus // O elemento recebeu o foco (False para Bubbles) blur // O elemento perdeu o foco (False para Bubbles)
Evento WebSocket
open // Conexão WebSocket estabelecida message // Mensagem recebida através do WebSocket error // A conexão WebSocket foi encerrada, com erro na transmissão / recepção de dados close // Conexão encerrada
Evento Histórico de sessão
pagehide // Quando o documento está para ser fechado e guardado no cache pageshow // Quando o documento é carregado pela primeira vez popstate // Quando o navegador carrega uma nova página ou restaura um estado salvo com History.
Evento Animações CSS
animationstart // A animação começou animationend // A animação acabou animationiteration // A animação se repete
Evento Transição CSS
transitionstart // A transição CSS começou (depois de algum delay) transitioncancel // A transição CSS foi cancelada transitionend // A transição CSS acabou transitionrun // A transição CSS começou (antes de qualquer delay).
Evento Formulários
reset // Botão de reset pressionado submit // Botão de submit pressionado
Evento Impressão
beforeprint // Caixa de diálogo de impressão está aberta afterprint // Caixa de diálogo de impressão está fechada
Evento Composição de texto
compositionstart // A composição de texto está preparada (semelhante ao keydown para uma entrada de teclado, mas funciona com outras entradas, como reconhecimento de fala). compositionupdate // Um caracter foi adicionado à trecho de texto que está sendo escrito compositionend // A composição de texto foi finalizada, ou cancelada.
Evento Tela
fullscreenchange // Um elemento alternou o modo de visualização para fullscreeen ou normal fullscreenerror // Não foi possível alternar para o modo fullscreen por problemas técnicos ou falta de permissão resize // A tela foi redimensionada scroll // A tela foi rolada (movimento de scroll)
Evento Área de transferência
cut // A seleção foi recortada e copiada para a área de transferência copy // A seleção foi copiada para a área de transferência paste // O item que estava na área de transferência foi colado
Evento Teclado
keydown // Qualquer tecla foi pressionada keypress // Qualquer tecla, com exceção de Shift, Fn e Caps Lock está pressionada (segurada) keyup // A tecla foi solta
Evento Mouse
auxclick // Um botão, ou dispositivo apontador (não primário) do dispositivo foi clicado e solto de um elemento. click // Um botão, ou dispositivo apontador (QUALQUER, mas em breve será apenas primário) foi clicado e solto. contextmenu // O botão direito do mouse foi clicado (antes do menu de contexto aparecer) dblclick // Um elemento foi clicado duas vezes em um intervalo de tempo curto mousedown // O mouse, ou dispositivo apontador está pressionando um elemento mouseenter // O mouse, ou dispositivo apontador está acima de um elemento que tem um listener ativo mouseleave // O mouse, ou dispositivo apontador foi removido do elemento com um listener mousemove // O mouse está se movendo enquanto está acima de um elemento com listener mouseover // O mouse foi movido para um elemento com um listener, ou em um de seus filhos mouseout // O mouse foi removido do elemento, ou de algum filho contendo um listener mouseup // O botão do mouse ou dispositivo foi solto pointerlockchange // O botão foi bloqueado ou liberado pointerlockerror // Não foi possível bloquear o dispositivo por motivo técnico ou de permissão select // Algum texto está sendo selecionado wheel // Foi detectada rotação no scroll do mouse
Evento Arrastar e soltar
drag // Um elemento ou uma seleção de texto está sendo arrastado. dragend // Uma operação de arrastar foi abortada (Por soltar o botão do mouse, ou apertando o Esc). dragenter // Um elemento ou texto selecionado foi arrastado para um local de destino válido dragstart // Uma operação de arrastar foi iniciada dragleave // Um elemento ou texto selecionado foi arrastado para fora de um local de destino válido dragover // Um elemento ou uma seleção de texto está sendo arrastado em uma área de destino válida. drop // Um elemento ou uma seleção de texto foi solto em um destino válido.
Evento Mídia
audioprocess // A entrada do buffer de ScriptProcessorNode está pronta para ser processada canplay // O navegador pode reproduzir o arquivo, mas estima que não haverá dados suficientes para reproduzir o arquivo sem interrupções para recarregar o buffer. canplaythrough // O navegador estima que poderá reproduzir o arquivo sem interrupções até o final. complete // A renderização de OfflineAudioContext foi finalizada. durationchange // O atributo duration foi atualizado. emptied // Ausencia de conteúdo. Por exemplo, este evento é enviado se a mídia foi carregada (ou parcialmente) e o método load() foi chamado para recarregar o conteúdo. ended // A reprodução foi finalizada devido ao fim do conteúdo loadeddata // O primeiro frame de mídia foi carregado. loadedmetadata // Os metadados foram carregados. pause // A reprodução foi pausada. play // A reprodução foi iniciada. playing // A reprodução está pronta para iniciar depois de ser pausada, ou atrasada devido a falta de dados. ratechange // A taxa de reprodução foi alterada. seeked // Operação de busca finalizada. seeking // Operação de busca iniciada. stalled // The user agent is trying to fetch media data, but data is unexpectedly not forthcoming. suspend // Media data loading has been suspended. timeupdate // The time indicated by the currentTime attribute has been updated. volumechange // The volume has changed. waiting // Playback has stopped because of a temporary lack of data.
Evento Progresso
abort // O progresso terminou (com sucesso). error // Erro no progresso. load // O progresso foi bem sucedido. loadend // O progresso foi interrompido (devido a 'error', 'abort' ou load) loadstart // O progresso iniciou progress // Em progresso timeout // O progresso foi finalizado pois o tempo pré determinado foi expirado.

Métodos de manipulação de string
let nome = "javaScript é a melhor linguagem de programação"; console.log(nome.lenght); // Saber quantos caracteres tem a string. console.log(nome.indexOf('Script')); // Procura a palavra especificada, retorna a posição que ela inicia (se não encontrar retorna -1) // Extrair informações da string. console.log(nome.slice(0, 10)); // Slice - Primeiro parametro posição inicial (obrigatório) e segundo parametro posição final. Tem a possibilidade de iniciar do final da strring informando números negativos. console.log(nome.substring(0, 10)); // substring - Igual o slice porém não pode começar do final. console.log(nome.substr(5, 3)); // Igual o slice, porém o segundo parametro é a quantidade de caracteres que quer pegar e não a posição final. console.log(nome.replace('de programação', '')); // replace - Pesquise pelo primeiro parametro e substitua ele pelo segundo console.log(nome.toUpperCase()); // Todas as letras maiúsculas. console.log(nome.toLowerCase()); // Todas as letras minúsculas. console.log(nome.concat(' ', 'atualmente')); // Para concatenação. Pode ter quantos parametros quiser. console.log(nome.trim()); // Remover espaços em branco inuteis no inicio e fim da string. console.log(nome.charAt(3)); // Saber qual o cartactere ta na posição informada. console.log(nome[3]); // Apartir do ECMAScript 5 da pra saber o caractere dessa forma, ao invés de usar a função charAt(). console.log(nome.split(' ')); // Transforma a string em array. Onde encontrar o que foi informado no parametro, divide a string.
